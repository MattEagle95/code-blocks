# features
- easy to use react material-ui
- manage multiple pm2 instances in one ui
- uses socket.io to listen for pm2 events
- user and role management (jwt)
-- supports multiple databases (sqlite3, postgresql, mysql)
-- auditlog
- notifications (email)
